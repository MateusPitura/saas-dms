## Description

A web-based, open-source Dealer Management System (DMS) that enables you to track vehicles, organise accounts, and generate reports

- [Features](#features)
- [How to Run](#how-to-run)
- [Technologies Used](#technologies-used)
- [Authors](#authors)

## Features

🚗 **Vehicle Tracking:** manage and monitor your vehicle inventory and details

💼 **Account Management:** organize accounts with clear records

📊 **Reporting:** generate reports to analyze sales and inventory

## How to Run

Access the site [https://dms.mateuspitura.com](https://dms.mateuspitura.com?utm_id=05RK83)
